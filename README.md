# 🏥 Healthcare Management System

A Java-based Healthcare Management System designed to automate hospital operations such as patient record management, doctor handling, billing, and administrative tasks. The system provides a secure and user-friendly interface with role-based authentication for efficient hospital management.

---

## 📌 Project Highlights

* Role-based login system (Admin/User)
* Designed based on DFD and ER Diagram
* Reduces manual paperwork and improves efficiency
* User-friendly Java Swing interface

---

## 🚀 Features

* 🔐 Secure Login System with Role-Based Access
* 👨‍⚕️ Doctor Management
* 🧑‍🤝‍🧑 Patient Record Management
* 💊 Medicine & Pharmacy Management
* 🧪 Lab Report Handling
* 🧾 Billing and Payment System
* 📊 Hospital Reports Management
* 🔑 Forgot Password Functionality

---

## 🧩 Modules

### 1. Administrator Module

* Manage doctors, patients, staff, and departments
* Monitor reports and hospital activities
* Manage user roles and permissions

### 2. Patient Module

* View appointments and prescriptions
* Access medical history and reports

### 3. Doctor Module

* Manage patient records
* Create prescriptions and appointments

### 4. Nurse Module

* Manage patient care and bed allocation
* Maintain operation and patient records

### 5. Pharmacist Module

* Manage medicine stock
* Provide medicines based on prescriptions

### 6. Lab Module

* Upload and manage diagnostic reports (X-ray, MRI, etc.)

### 7. Accountant Module

* Generate invoices
* Manage payments and billing records

---

## 🛠️ Technologies Used

* **Frontend:** Java (Swing, AWT)
* **Backend:** MySQL / Oracle
* **Connectivity:** JDBC
* **Platform:** Windows

---

## 📊 System Workflow

1. User logs into the system
2. System checks credentials
3. Role-based access is verified
4. User is redirected to respective module
5. Data is managed and stored efficiently
